## General Information
-  A simple console based Java combat game where you, the player, must face a computer opponent. The player can select from five different characters, each highlighting their own unique combat and/or healing abilities. Each turn gives the player the option to move, change the direction they are facing, or engage in combat. The computer players selection is chosen randomly. The game is won when the player successfully strikes the opponent during combat. The players move around a 6x6 grid and damage is only dealt within a certain distance of each character.
